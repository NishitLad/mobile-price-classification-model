## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost

## Dataset
- train.csv
- test.csv

## Features
- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Model Training
- Price Range Prediction
